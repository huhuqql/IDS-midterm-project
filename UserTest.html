User Test

We have modified our design for the Impact bar chart and Number of Incidents trend graph based on the feedback we received from users. Initially, the interaction components for these two graphs were designed as follows: we sorted the countries based on the total number of terrorist attacks from 1970 to 2016 and selected the top ten to be displayed on the menu. the user will be prompted to choose from the top ten countries and the graph will only display data that pertains to the selected country (see Figure 1).

However, some of the users expressed that they would like to have the option to explore countries outside of the top ten list. Therefore, we replaced the top-ten-countries menu with a search bar function. Now when the user clicks on the button “Choose Another Country”, he/she will be prompted to search for any country he/she will like to explore (see Figure 2).

In addition, we also made some minor tweaks to the graphs including inserting legends, changing layouts and adjusting font size based on the user feedback.
